## 변경 사항

### Before (이전) ❌
```
퇴직과 관련된 정보는 문서에서 다음과 같이 언급되어 있습니다:
자동퇴직의 경우:
정년에 도달했을 때
사망한 경우
...
```
- 단순 텍스트 나열
- 구조 없음
- 가독성 낮음

### After (개선 후) ✅
```markdown
## 📋 퇴직 관련 정보

### 핵심 요약
- **자동퇴직**: 정년 도달, 사망 등
- **의원퇴직**: 본인 의사로 제출
- **퇴직원 제출**: 2개월 전

### 📌 상세 내용

#### 1. 자동퇴직
- **정년**: 60세 도달 시
- **사망**: 당일 퇴직
- [출처: 인사관리규정.pdf, p.15]

#### 2. 의원퇴직
...
```

## 📝 답변 예시

### 예시 1: 퇴직 정보 질문

**질문**: "26살 3년차 직원의 연차는?"

**Before**:
```
3년차 직원의 경우 연차 휴가는 근로기준법에 따라 15일입니다. 1년 미만은 월차를 사용하고...
```

**After**:
```markdown
## 📋 3년차 직원 연차 정보

### ✅ 답변
**3년차 직원의 연차**: 15일

### 📌 상세 내용

#### 근속연수별 연차
| 근속 | 연차 일수 |
|------|----------|
| 1년 | 15일 |
| 3년 (해당) | 15일 |
| 5년 | 16일 |
| 10년 | 20일 |

#### 사용 규칙
- **사전 신청**: 3일 전까지
- **승인**: 팀장 결재 필요
- **이월**: 다음 해 가능

[출처: 인사관리규정.pdf, p.45]
```

### 예시 2: 복잡한 규정 질문

**Before**:
```
퇴직금은 계속 근로연수 1년에 대하여 30일분 이상의 평균임금...
```

**After**:
```markdown
## 💰 퇴직금 계산 방법

### 📌 기본 공식
```
퇴직금 = 1일 평균임금 × 30일 × 근속연수
```

### 계산 예시

**근속 3년, 월급 300만원인 경우:**

1. **1일 평균임금**: 100,000원
2. **30일분**: 3,000,000원
3. **근속 3년**: × 3
4. **퇴직금**: **9,000,000원**

### ⚠️ 주의사항
- 1년 미만 근속 시 지급 제외
- 평균임금은 최근 3개월 기준

[출처: 퇴직급여규정.pdf, p.12]
```
