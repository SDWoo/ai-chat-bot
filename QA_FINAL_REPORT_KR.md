## 🐛 발견 및 수정된 버그

### 버그 #1: 백엔드 API 문법 오류 (Critical, ✅ 수정 완료)

**파일**: `backend/app/api/routes/chat.py`  
**라인**: 31, 45, 66

#### 문제
Python f-string 내에서 여러 줄에 걸친 딕셔너리 사용 시 `SyntaxError: unterminated string literal` 발생

#### 원인
```python
# ❌ 잘못된 코드
yield f"data: {json.dumps({
    'type': 'metadata',
    'conversation_id': conversation_id,
    'sources': sources,
})}\n\n"
```

#### 해결
```python
# ✅ 수정된 코드
metadata = {
    'type': 'metadata',
    'conversation_id': conversation_id,
    'sources': sources,
}
yield f"data: {json.dumps(metadata)}\n\n"
```

#### 영향 범위
- ❌ 수정 전: 백엔드 시작 불가
- ✅ 수정 후: 정상 작동

#### 검증
